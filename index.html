HTML5 Structural Elements

Posted on December 20, 2011

This page was built using HTML5. The real magic behind this HTML5 page is that each structural element accurately reflects the purpose of the data contained within the element. This particular section is contained within an article element which represents an article of data.

Don't think of "article" as a news article, think an article of clothing - a discrete item.
- Introducing HTML5

Why is this important? Because each structural element is specific to its prupose, it would be easier to parse and understand the content. This might not mean much to you, but this page supports web accessibility. A third party reader could very easily parse through the markup and find all content contained within an article element. In the case of this article, the parser would find a header that explains the content followed by each paragraph within the article and, finally, a footer for the element with information about the person that posted this information.
